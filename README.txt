# Pushing files from computer to github
1. Initialize working directory from your computer to make it repository
   git init -- run from working directory folder
2. login to github.com and create new Repository 
3. Use below set of commands to push your code to github repo
   git add README.txt
   git commit -m "First Commit"
   git remote add origin https://github.com/VivekNavgale/SecondProject.git
   git push origin master
4. Use below command frequently
   git status
